# Návrhy blogov podľa produktov + odporúčané obrázky

Tento súbor obsahuje pre každý tvoj produkt:
- návrh 2–3 blogových článkov,
- stručný popis (čo má byť obsahom),
- odporúčané obrázky: **1 hlavný** a **2–4 vedľajšie**,
- ku každému obrázku ALT text + odkaz na vhodný zdroj (Unsplash – free, bez potreby atribúcie).

## 01 – Tvorba webov  
**Moderné a responzívne webové stránky postavené na najnovších technológiách**

### 1.1 Prečo je v roku 2025 responzívny web povinná jazda, nie bonus

**Stručný obsah článku**  
- Vysvetliť, že väčšina návštevnosti je dnes z mobilu.  
- Mobile-first indexing: Google hodnotí primárne mobilnú verziu webu.  
- Rozdiel medzi starou „desktop + mobilná verzia“ a moderným responzívnym webom.  
- Dôsledky: horšie pozície, vyšší bounce rate, menej dopytov.  
- Ako to riešiš ty pri tvorbe webov (technológie, testovanie na zariadeniach).

### 1.2 Ako rýchlosť webu ovplyvňuje počet dopytov a telefonátov

**Stručný obsah článku**  
- Prepojenie: rýchlosť webu → spokojnosť používateľov → SEO → dopyty.  
- Štatistiky o tom, koľko ľudí odíde pri pomalom načítaní.  
- Najčastejšie chyby: obrovské obrázky, slabý hosting, zlé šablóny.  
- Core Web Vitals – jednoduché vysvetlenie pre ne-IT človeka.  
- Príklad „pred a po“ – ako zrýchlenie webu prinieslo viac konverzií.
